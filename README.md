# Node.js HTTP сервер (учебная работа)

## О проекте

Это простой HTTP сервер, сделанный на Node.js с использованием встроенного модуля `http`.

Сервер обрабатывает несколько маршрутов и отдаёт простые HTML страницы.

---

## Как запустить

Сначала нужно установить зависимости :

```bash
npm install
```

Дальше запускаем сервер:

```bash
node server.js
```

или

```bash
npm start
```

После запуска сервер будет доступен здесь:

```bash
http://localhost:3000
```

Если нужно, можно поменять порт:


```bash
PORT=4000 node server.js
```

---

## Маршруты

- `/` — главная страница (приветствие)
- `/about` — информация о проекте
- `/contact` — страница для связи

Если открыть любой другой адрес, появится:

Page Not Found


---

## Что используется в проекте

- Node.js
- встроенный модуль `http`
- простая ручная маршрутизация

Без Express и без сторонних библиотек.
